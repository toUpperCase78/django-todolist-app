About This Project

This project was created with Python language and Django framework, which enables to simulate the well-known to-do list application.

To carry out this project into the runnable condition at the most reasonable level, I benefited from this YouTube video by Tech With Tim, teaching Django with various aspects, which took 3+ hours to complete.

The video featured these parts about Django, as well as other things for how the related components work:

  • SQLite3 Database
  • Admin Dashboard
  • Templates & Custom HTML
  • Simple & Custom Forms
  • Adding Sidebar
  • Adding Bootstrap
  • User Registration
  • Login, Logout, User Authentication
  • User Specific Pages

Later on, I continued improving this project to provide richer user interface, enable deleting items from to-do lists, and implement fully functional user registry and login / logout procedure.

Hopefully, this will pave the way for creating web applications using Python & Django with other popular libraries alongside, for unique user experience and interactions and being alternative to other web frameworks such as React, TypeScript, etc.
